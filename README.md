# GoBarber
:barber: Sample API REST built with NodeJS, Express and Sequelize

## Running Locally

Rename ".env.example" to ".env" 

```sh
yarn

# start the server
yarn dev

# start the queue
yarn queue
```

Runs the app in development mode.<br>
Open [http://localhost:3333](http://localhost:3333) to view it in the browser.
